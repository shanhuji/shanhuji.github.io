2011年11月19日

发表于 2011-11-19 更新于 2022-05-29 分类于 若有所思 本文字数： 410 阅读时长 ≈ 1 分钟

今天一位久别的同学来杭，另找了2个同学一起小聚

席间大家聊到近况，感慨颇多：

一是，人的惰性变得很重，年龄大了，记忆减退，很多过往的好朋友突然发现居然有快十年都没有联系过了，昔日的影像也变得越来越模糊，这位同学突然打来电话，一时间竟然没反应过来。。。。。

二是，大家的虽然同窗共度，可是大学之后，就开始各奔东西了，几年下来，各人的际遇的差别可谓很大，大部分同学还是留在了家乡，从事的工作也多种多样，亦多以结婚生子，而漂泊在外乡的却还有几个单身依旧，而且事业也不太稳定。

三是，生活里挺多的不如意，时代不同了，貌似大家都对未来有种很强的不确定因素，不知道未来的路该如何走，这个跟当日在高中满怀着很多理想是很不相同的，其间也有聊到换工作的问题，大家似乎都有这么的想法。。。。。

总的说来，他乡遇故知，还是很开心的有件事情，现在发现，原来人的一生中能够认识的人虽然很多很多，但是能够谈得来的朋友却真的是不太多，而能够可谓知心的却更少了！

在浙江省图书馆一楼大厅的墙壁上，有四块名言牌，其中大哲学家黄宗羲的一块读来使人唏嘘不已。

大丈夫行事，论是非，不论利害；论顺逆，不论成败；论万世，不论一生。
—（明）黄宗羲
